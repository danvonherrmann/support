TV says "No Signal" or black screen‚ñº

Check Power First:

  • Make sure the cable box is powered on (look for lights on the front)
  • Check all light switches near the TV
  • If TV powers on, check the power strip or outlet for the cable box

Check TV Input:

  • Press INPUT button on your remote or TV
  • Try changing inputs manually with TV remote or button on TV
  • Look for HDMI1, HDMI2, or similar labels
  • The correct input should show the cable box screen

Check Connections:

  • Make sure all cables are firmly connected behind the TV and box
  • Check that HDMI cable is securely plugged in both ends

Restart the Box:

  • Unplug the cable box for 30 seconds
  • Plug it back in
  • Wait 3-5 minutes for the box to fully restart
‚ö†Ô∏è See an error message? Messages like "Complete Signal Loss" or "Programming Not Authorized" require technical support. Call 251-967-4455 immediately.
Remote isn't working‚ñº

Check Batteries:

  • Check that batteries are installed correctly (+ and - aligned)
  • Try replacing with fresh batteries

Check Remote Mode:

  • Press the SAT button on the side/top of the remote
  • Make sure you're in satellite mode, not TV mode
  • The SAT mode light should be on

Check Line of Sight:

  • Point the remote directly at the cable box (not the TV)
  • Make sure nothing is blocking the front of the cable box
  • Remove any objects between remote and box
üí° Pro Tip: The buttons on the SIDE or TOP of the remote (TV, INPUT, SAT) are the most important! Press SAT to control the cable box, TV to control the TV.

Still not working? Try these:

  • Unplug the cable box for 30 seconds and plug back in
  • Test if the TV remote works (rules out TV issues)
  • If nothing works, call 251-967-4455 for replacement
No sound or audio issues‚ñº

Quick Fix - Power Cycle:

  • Unplug both the TV and cable box
  • Wait 30 seconds
  • Plug both back in
  • If volume returns to normal, you're all set!

Sound works on TV apps but not cable box?

  • Unplug the HDMI cable from both the TV and cable box
  • Try plugging it into a different HDMI port on the TV
  • Make sure both ends are firmly connected

Still no sound?

  • Try a different HDMI cable if available
  • Check TV volume settings and mute button
  • If nothing works, call 251-967-4455
üí° Note: If sound works for Netflix/apps but not the cable box, it's usually an HDMI connection issue, not the box itself.

WiFi Connection Issues‚ñº

First, determine the issue:

  • Is it one device or multiple devices that cannot connect?
  • Do you see the correct unit number or SSID on the list of available connections?
  • Verify the SSID/network name that you are trying to connect to

Single Device Issues:

  • Verify password is entered correctly (check CAPS LOCK is OFF)
  • Try "forgetting" the network and reconnecting
  • Restart the device

Multiple Devices or No Connection Available:

  • Check if router has power (look for lights)
  • Try unplugging router for 30 seconds, then plug back in
  • Wait 2-3 minutes for router to fully restart
  • Try connecting again
